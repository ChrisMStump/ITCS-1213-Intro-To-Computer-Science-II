PROJECT TITLE: Programming Test 2
PURPOSE OF PROJECT: To demonstrate understanding of PrintWriter, String conversion and input files.
VERSION or DATE: 3-01-2015

This program takes in a text file as input. If the text contains numerical values,
the program changes them to the english equivalent and saves the new text in a file
called outSentence.txt

For Example: 1 would be One. 8 would be Eight.
