The program reads from the clientdata.txt and creates Human and Pet objects. Then,
the program displays each human with their pets.

Human: Last, First, Address, ZIP, Num Visits, Balance, Num Pets
Each pet has there own line below the human.
Pet: Name, Type, Weight, Last Rabies Shot, Last Visit
