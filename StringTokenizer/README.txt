PROJECT TITLE: Programming Test 3
PURPOSE OF PROJECT: To demonstrate understanding of String Tokenization and manipulating data.
VERSION or DATE: 4-1-2015

This program does not take any user input. The program starts with a date in String
format such as "April 1, 2015"

The program breaks the String into tokens and stores them into month, day and year
respectively.

The program can also add and subtract days and report the new date in both short and
long form.
SHORT MM/DD/YYYY
LONG MONTH DAY, YEAR
